# Medical Cost Prediction

End-to-end machine learning project to predict individual medical insurance costs using regression techniques.

## Project Structure
See folders for data, source code, configs, and outputs.

## ML Techniques
- Feature engineering
- Regression models
- Hyperparameter tuning (Optuna)
- Explainability (SHAP)

## How to Run
1. Create venv
2. Install requirements
3. Run training script
